# Че делать

1. Скинуть все исходники, кроме тестов, в папку `src`
2. Скинуть все тесты в папку `test`

# Как прогнать тетсты на Mac и Linux, если у вас нет CLion

`$ ./run_tests.sh`

# Как прогнать тесты на Windows

`> cmake .`

`> make dz_test`

`> test\dz_test` || `> test\dz_test.exe`
